## **overview**

**dendata** The goal of dendata is provides the dengue spatial databases (areal
data) available and published in scientific journals. The package included the next datasets. 

  - **`den_loc_mex()`** def.
  - **`ajmrzik16_col()`** def.
 


## Installation

You can install the released version of dendata from
[CRAN](https://CRAN.R-project.org) with:

``` r
install.packages("dendata")
```

And the development version from [GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("fdzul/dendata")
```
